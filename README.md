# Argentina Investment Visa

Landing page for [Argentinainvestmentvisa.com](https://argentinainvestmentvisa.com) — Argentina Citizenship by Investment program. Built with Astro and Tailwind CSS.

## Commands

| Command           | Action                          |
| :---------------- | :------------------------------ |
| `npm install`     | Install dependencies            |
| `npm run dev`     | Start dev server at localhost:4321 |
| `npm run build`   | Build for production to `./dist/` |
| `npm run preview` | Preview production build       |

## Project structure

- `src/pages/` — Routes (index)
- `src/components/` — Navbar, Hero, TrustBar, About, Features, Gallery, FAQ, LeadCapture, Footer
- `src/layouts/` — Layout with meta and fonts
- `src/styles/` — Global CSS and design tokens
- `public/` — Favicon and static assets

Form submissions are handled by [Formcarry](https://formcarry.com).
